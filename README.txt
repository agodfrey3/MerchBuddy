#README

This is a project which scrapes the item price data from each tradeable item in Runescape and places it into a MYSQL Database, which is then analyzed to find the best item(s) for merching, as well as other various information that can be drawn from price trends. 


Languages used: 
Python(2.7)
Python(3.5)
